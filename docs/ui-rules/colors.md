# Colors

Defined in `src/main/webapp/css/app.css` as CSS variables on `:root`.

| Token | Value | Use |
|-------|-------|-----|
| `--color-primary` | `#2563eb` | Links, logo, primary buttons, accents |
| `--color-primary-dark` | `#1d4ed8` | Primary hover |
| `--color-danger` | `#dc2626` | Destructive actions, errors, expired |
| `--color-success` | `#16a34a` | Success badges, correct answers |
| `--color-warning` | `#d97706` | Timer warning, in-progress |
| `--color-bg` | `#f8fafc` | Page background |
| `--color-surface` | `#ffffff` | Cards, header, footer |
| `--color-border` | `#e2e8f0` | Borders / dividers |
| `--color-text` | `#1e293b` | Body text |
| `--color-muted` | `#64748b` | Secondary text |

Prefer these variables over hard-coded hex in new UI.
